# **Prandium**

*Simple menu-to-JSON converter - digitize any restaurant menu in seconds*

---

## 1. Executive Summary

Prandium is a **simple web app** that lets restaurants and food businesses turn any menu (PDF, image, or text) into a clean, structured JSON file. Upload your menu, get a digital version ready for online ordering, delivery apps, or your website. No technical skills required.

---

## 2. Problem Statement

| Challenge | Impact | Current Solutions |
|-----------|--------|-------------------|
| **Paper menus** | Can't use for online ordering | Manual data entry |
| **Messy formats** | Images, PDFs, and text are hard to digitize | Expensive services |
| **No easy tool** | Small restaurants can't afford custom dev work | DIY, error-prone |

---

## 3. Solution Overview

A **simple web app** that:
- Accepts menu uploads (PDF, image, or text)
- Extracts menu items, prices, and categories
- Outputs a clean, structured JSON file
- Lets users review and edit before export
- No registration required for basic use

---

## 4. Key Features (MVP - Week 1)

| Feature | Description | Implementation Time |
|---------|-------------|-------------------|
| **Menu Upload** | Upload PDF, image, or text | 3 hours |
| **OCR & Parsing** | Extract items, prices, categories | 6 hours |
| **Review/Edit** | User can edit extracted data | 4 hours |
| **Export JSON** | Download structured menu | 2 hours |

---

## 5. Example Workflow

1. **Upload** - Drag and drop menu file
2. **Extract** - App parses items, prices, categories
3. **Review** - User edits/corrects as needed
4. **Export** - Download JSON file

---

## 6. Technical Stack (Simple)

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Simple server (Python/Flask)
- **OCR**: Tesseract or similar open-source tool
- **Hosting**: Netlify/Vercel (free tier)
- **Domain**: $10/year
