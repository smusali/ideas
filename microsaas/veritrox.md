# **Veritrox — Simple Feedback Analyzer**

*A web app that analyzes customer feedback and gives you instant insights. Upload feedback from surveys, reviews, or support tickets, and get labeled topics, sentiment analysis, and action suggestions. Perfect for small businesses and startups.*

---

## 1. Executive Summary

**Veritrox** is a simple feedback analysis tool that helps you understand what customers are saying. Upload any feedback (surveys, reviews, support tickets), and get instant insights with labeled topics, sentiment scores, and suggested actions. No data science skills needed - just upload and get actionable insights in seconds.

---

## 2. Problem Statement

People struggle with customer feedback because:
- **Too much data** - Hundreds of responses to manually review
- **No structure** - Feedback scattered across different sources
- **Slow analysis** - Takes days to identify patterns and trends
- **Missed insights** - Important issues get buried in the noise
- **No action plan** - Don't know what to do with the feedback

---

## 3. Solution Overview

Veritrox provides instant feedback analysis:
1. **Upload Feedback** - Paste text or upload CSV files
2. **Get Analysis** - Automatic topic labeling and sentiment detection
3. **See Insights** - Clear breakdown of what customers are saying
4. **Take Action** - Suggested next steps based on feedback

**MVP Features:**
- Text input and CSV upload
- Automatic topic detection (bugs, features, pricing, etc.)
- Sentiment analysis (positive/negative/neutral)
- Action suggestions
- Export results to CSV

---

## 4. User Workflow

1. **Upload** - Paste feedback text or upload CSV file
2. **Analyze** - Click to get instant analysis
3. **Review** - See topics, sentiment, and suggestions
4. **Export** - Download results for your team

**Example Results:**
```
📊 Feedback Analysis

Topics Found:
• Bug Reports (15 mentions) - Login issues, slow loading
• Feature Requests (8 mentions) - Dark mode, mobile app
• Pricing Questions (5 mentions) - Plan confusion, billing

Sentiment: 65% Positive, 25% Negative, 10% Neutral

Suggested Actions:
• Fix login bugs (high priority)
• Add dark mode to roadmap
• Clarify pricing page content
```
