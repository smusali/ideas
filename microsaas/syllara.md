# **Syllara**

*Simple worksheet and quiz generator for tutors - create custom practice in seconds*

---

## 1. Executive Summary

Syllara is a **simple web app** that lets tutors and teachers generate custom worksheets and quizzes for any topic, instantly. Enter a subject and level, get printable or digital practice sets. No registration required for basic use.

---

## 2. Problem Statement

| Challenge | Impact | Current Solutions |
|-----------|--------|-------------------|
| **Manual worksheet creation** | Takes hours, repetitive | Copy-paste, templates |
| **No customization** | Hard to tailor to each student | Generic PDFs |
| **No easy tool** | Existing tools are complex or expensive | DIY, error-prone |

---

## 3. Solution Overview

A **simple web app** that:
- Lets users enter a topic, grade, and question type
- Instantly generates a worksheet or quiz (MCQ, fill-in, short answer)
- Allows editing before export
- Exports as PDF or Word
- No registration required for basic use

---

## 4. Key Features (MVP - Week 1)

| Feature | Description | Implementation Time |
|---------|-------------|-------------------|
| **Topic Input** | Enter subject, grade, question type | 2 hours |
| **Question Generation** | Auto-generate questions | 6 hours |
| **Edit/Review** | Edit questions before export | 3 hours |
| **Export** | Download as PDF/Word | 3 hours |
| **Basic UI** | Clean, mobile-friendly design | 4 hours |

**Total: 18 hours (3 days)**

---

## 5. Example Workflow

1. **Input** - Enter "Algebra, Grade 7, 10 MCQs"
2. **Generate** - App creates 10 questions
3. **Edit** - User reviews/edits as needed
4. **Export** - Download as PDF or Word

---

## 6. Technical Stack (Simple)

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Simple server (Python/Flask)
- **Question Generation**: Open-source LLM or prompt API
- **Hosting**: Netlify/Vercel (free tier)
- **Domain**: $10/year
