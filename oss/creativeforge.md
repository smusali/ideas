# **CreativeForge** — *Simple Text Art Generator*

*A lightweight, open-source tool that helps you create beautiful text-based artwork with minimal effort.*

---

## **What is CreativeForge?**

CreativeForge is a simple web application that lets you create text-based artwork, ASCII art, and simple graphics. Built for developers and creators who want to generate visual content without complex design tools.

---

## **Core Features (MVP - 7 Days)**

### **Day 1-2: Basic Setup**
- Simple web interface for creating text art
- Basic database to store artwork
- User registration and login

### **Day 3-4: Core Functionality**
- Create text-based artwork with custom fonts
- Basic color and style options
- Save and organize your artwork
- Simple gallery showing your creations

### **Day 5-6: Enhanced Features**
- Export artwork as images (PNG, SVG)
- Share artwork via links
- Basic templates and presets
- ASCII art generator

### **Day 7: Polish & Deploy**
- Responsive design for mobile
- Deploy to free hosting platform
- Write documentation and README

---

## **Simple Data Model**

```
User:
- id, email, password_hash, created_at

Artwork:
- id, user_id, title, text_content, style_config, created_at

Style Config:
- font_family, font_size, color, background_color, alignment
```

## **Why Open Source?**

- **Transparency**: You can see exactly how your artwork is generated
- **Customization**: Modify it to fit your specific creative needs
- **Learning**: Great project for developers to learn web development
- **Community**: Others can contribute features they want
- **Sharing**: Creators can share and collaborate on artwork
